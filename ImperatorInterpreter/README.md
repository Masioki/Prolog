#  Imperative language interpreter in Prolog  

Variables are uppercase  
  VAR := 1;  

Equal      =  
Not equal /=  
And       and  
Or        or  
Greater   >  
Less      <  
GE        >=  
LE        =<  
Modulo    mod  
  
Loop:  
    while condition do  
    ...  
    od;  
      
If statement:  
    if condition then  
    ...  
    else  
    ...  
    fi;
